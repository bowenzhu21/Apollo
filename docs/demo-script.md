# SPATIAL Demo Script

Target length: 45-60 seconds.

1. Start on the boot screen and initialize SPATIAL.
2. Enable vision and show the camera skeleton tracking.
3. Raise the right hand open palm to queue prompt 1.
4. Hold left-hand pinch for 2 seconds to send.
5. Show the model response and latency counter.
6. Raise right-hand three fingers to queue a plan prompt.
7. Hold left-hand pinch again to send.
8. Toggle debug mode briefly to show gesture telemetry.
9. Hold left-hand fist for 2 seconds to clear the conversation.

Narration:

> SPATIAL is a hands-free AI interface. MediaPipe tracks gestures locally in the browser, while model access runs through a server-side API route so keys are never exposed. Left-hand gestures are held for two seconds to prevent accidental actions. Right-hand gestures queue editable prompts, and the backend can switch between Gemini and OpenAI providers through environment variables.
